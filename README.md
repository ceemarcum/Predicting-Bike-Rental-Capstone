# Predicting Bike Rental 

## Overview
This project predicts hourly bike rental demand by preparing the dataset and exploring rental patterns through correlation analysis and visualizations. After encoding categorical variables and scaling numerical features, the data was split into training and testing sets. Linear, Ridge, and Lasso regression models were then built and evaluated to determine how well each approach predicts rental counts.

## Key Visualizations
- **Histograms:** Showed the distribution of key numerical variables, including rental counts and weather measurements.
- **Heatmap:** Highlighted correlations among weather conditions, time features, and bike rental demand.
- **Box Plots:** Compared rental count distributions across categorical features such as holidays, seasons, and weekdays, helping identify patterns and outliers.

## Why It Matters
These visualizations turn raw bike rental data into clear, actionable insights that help explain how time, weather, and seasonal patterns influence demand. By understanding these relationships, we can better anticipate rental needs, improve resource planning, and support data driven decisions for managing bike availability.
